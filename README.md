# Team Task Manager

## Features
- JWT Authentication
- Project Management
- Task Management
- Dashboard Analytics
- Protected Routes

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- Tailwind CSS

## Installation

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Deployment
- Backend: Railway
- Frontend: Vercel
